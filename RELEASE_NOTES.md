## Open in Shell

You can now open any folder directly in the remote shell console.

---

### Features

- right-click on a folder → open it in the shell
- right-click on a file → open its parent folder in the shell
- reuses existing shell window (no duplicates)
- automatically navigates to the selected path using `cd`

---

### Improvements

- better integration between file explorer and remote shell
- faster workflow when navigating and debugging the MIB filesystem

---

### Notes

This feature builds on the remote shell console introduced in v0.2.1 and improves usability by linking both tools together.
